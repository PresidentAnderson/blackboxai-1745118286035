# Booking Sign-In Page

## Project Overview
The Booking Sign-In Page is a simple and responsive web interface for user authentication. It allows users to sign in with their email or via popular social media options like Google, Apple, and Facebook. The page is styled using TailwindCSS for modern aesthetics and responsive design.

## Installation
To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/booking-signin-page.git
   ```
   
2. **Navigate to the project directory**:
   ```bash
   cd booking-signin-page
   ```

3. **Open the HTML file**:
   Open `booking-signin.html` in your web browser.

## Usage
- Users can enter their email address to sign in or create a new account.
- The page also provides options to sign in using Google, Apple, or Facebook.
- Users can manage cookie settings by accepting or declining cookies based on their preferences.

## Features
- Responsive layout that adjusts to different screen sizes.
- User-friendly interface for signing in or creating an account.
- Options to sign in with popular social media accounts.
- Cookie management settings allowing users to accept or decline cookies.
- Smooth fade-in animation for a modern touch.

## Dependencies
This project uses the following external libraries:
- [TailwindCSS](https://tailwindcss.com/): A utility-first CSS framework.
- [Font Awesome](https://fontawesome.com/): For social media icons.

These libraries are linked directly from their respective CDNs in the HTML file.
